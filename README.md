# BE Custom Queue Bot
This is the custom queue bot made for the BE community. Originally, this was supposed to help with organizing customs, but over time it has turned into a project to help me learn how to code and how to set up a coding environment. This isn't really set up for public use right now, but maybe i'll change that in the future. 

PS some things aren't specified in the .env file (because stupid brain) and you'll have to manually go through the code and change it yourself. Everything regarding this is in cqb.py, viewphrases.py, and removephrases.py. Pretty easy changes tho so you'll be fine.
